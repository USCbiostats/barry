class BArray$<$ Cell\+\_\+\+Type, Data\+\_\+\+Type $>$}

Baseline class for binary arrays. 

{\ttfamily \mbox{\hyperlink{class_b_array}{BArray}}} class objects are arbitrary arrays in which non-\/empty cells hold data of type {\ttfamily Cell\+\_\+\+Type}. The non-\/empty cells are stored by row and indexed using {\ttfamily unordered\+\_\+map}s, i.\+e. {\ttfamily std\+::vector$<$ std\+::unordered\+\_\+map$<$size\+\_\+t,Cell\+\_\+\+Type$>$ $>$}.
