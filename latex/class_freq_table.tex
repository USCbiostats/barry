\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T = double$>$\newline
class Freq\+Table$<$ T $>$}

Frequency table of vectors. 

This is mostly used in {\ttfamily \mbox{\hyperlink{class_support}{Support}}}. The main data is contained in the {\ttfamily data} double vector. The matrix is stored in a row-\/wise fashion, where the first element is the frequency with which the vector is observed.

For example, in a model with {\ttfamily k} terms the first k + 1 elements of {\ttfamily data} would be\+:


\begin{DoxyItemize}
\item weights
\item term 1
\item term 2
\item ...
\item term k 
\end{DoxyItemize}
