\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_flock}{Flock}} is a group of \mbox{\hyperlink{class_geese}{Geese}}. 

This object buils a model with multiple trees (\mbox{\hyperlink{class_geese}{Geese}} objects), with all of these using the same Phylo\+Model object. Available counters (terms) can be found in counter-\/phylo.
