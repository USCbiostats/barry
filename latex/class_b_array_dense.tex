Baseline class for binary arrays. 

{\ttfamily \mbox{\hyperlink{class_b_array_dense}{BArray\+Dense}}} class objects are arbitrary arrays in which non-\/empty cells hold data of type {\ttfamily Cell\+\_\+\+Type}.
